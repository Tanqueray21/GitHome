# javaweb-task

#### 介绍
Servlet和JSP课程任务单项目结构模板，参训学员使用此 maven项目结构来编写自己的JAVAWEB课程【Servlet和JSP】的任务单代码。

#### 软件架构
```
javaweb-task/
............|javaweb-task1/    ---> 存放Servlet技术开发第一个任务单的代码
..........................src/
..........................pom.xml
............|javaweb-task2/    ---> 存放Servlet技术开发第二个任务单的代码
..........................src/
..........................pom.xml
............|javaweb-task3/    ---> 存放Servlet技术开发第三个任务单的代码
..........................src/
..........................pom.xml
............|javaweb-task4/    ---> 存放Servlet技术开发第四个任务单的代码
..........................src/
..........................pom.xml
............|javaweb-task5/    ---> 存放Servlet技术开发第五个任务单的代码
..........................src/
..........................pom.xml
............|javaweb-jsp-task6/    ---> 存放Servlet与JSP技术开发第一个任务单的代码
..............................src/
..............................pom.xml
............|javaweb-jsp-task7/    ---> 存放Servlet与JSP技术开发第二个任务单的代码
..............................src/
..............................pom.xml

............|pom.xml    --->最外层的pom.xml文件
```


#### 安装教程

1. 在本地工作区目录下，执行 git clone https://gitee.com/bogetang/javaweb-task.git 拉下项目到本地工作区。
2. 根据自己所要开发的任务单编号，进入相应的模块中开发

#### 使用说明

1. 本人任务单项目开发好后，自己先进行单元测试
2. 做个人的项目整体测试
3. 执行mvn clean，把 target/ 目录清除
4. 把项目文件夹通过钉钉发给老师。
